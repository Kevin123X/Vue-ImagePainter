# Vue image-painter
图片涂鸦、绘制、标注
## 支持功能
1.画笔颜色🎨

2.画笔粗细✏️

3.画布放大🔍

4.画布缩小🔍

5.清除画布🧹

6.保存图片🏔

7.回退一步↩️

8.前进一步↪️

## 画笔工具类型
1.画笔✏️

2.直线📏

3.圆形 ○

4.矩形🔲

5.橡皮🧽

6.文字📝


## Project setup
```
npm install
```
